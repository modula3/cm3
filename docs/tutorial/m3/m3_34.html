Looping Statements

Modula-3 provides the Pascal loop constructs FOR, WHILE and REPEAT, as well as the LOOP statement, which loops infinitely. In addition, the EXIT statement can appear in any loop construct, and causes the innermost loop to terminate when it is executed.

  • FOR Statement: Iterating through fixed loops.
  • WHILE Statement Loop while a condition holds.
  • REPEAT Statement: Repeat until a condition no longer holds.
  • LOOP Statement Loop forever, or until EXIT is called.
  • EXIT Statement: Exit body of the current loop.
