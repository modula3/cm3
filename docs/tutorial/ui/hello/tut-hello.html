Hello world

The first Trestle program we present is the classic ``Hello World'' program. The program Hello.m3 illustrates the basics of how to install a VBT as an independent portion on the user's display device. The program uses a simple unstructured VBT called a TextVBT. A TextVBT displays a string in a window of its own. By default VBT's are sized to the smallest possible size containing the text.

The Trestle subprocedure Install used in the program Hello.m3 manages all the details in creating a window on the screen, in this case, the TextVBT. Additional arguments to Install allow one to change the default window title, icon title, etc.

The Trestle subprocedure AwaitDelete is the main event loop of the program. The Modula-3 program Hello.m3 does not terminate nor does it accept any input from the user. The text ``Hello World'' is displayed until the program is terminated by some external mechanism, for instance, if the process is killed.
