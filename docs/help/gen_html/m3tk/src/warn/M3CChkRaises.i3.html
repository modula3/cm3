IMPORT AST, M3AST_AS;
IMPORT ASTWalk;
The following procedures check whether procedure calls or RAISE statements raise exceptions which are not dealt with by an enclosing RAISES or TRY EXCEPT statement, or mentioned in an FATAL pragma. 

TYPE
  Handle <: ASTWalk.Closure;

PROCEDURE NewHandle(
    cu: M3AST_AS.Compilation_Unit)
    : Handle
    RAISES {};
Create a new handle for use with the 'Node' procedure. 'cu' is the unit containing the tree which will be walked. The result handle can be passed directly to the tree walker or directly to 'Node' from another tree walk, as desired. 

PROCEDURE Node(h: Handle; n: AST.NODE; vm: ASTWalk.VisitMode) RAISES {};
This procedure is intended to be called from the tree walker (in entry and exit mode) so that it is called on every node in a section of AST. It checks for places which may raise exceptions which may not be caught. 

END M3CChkRaises.
