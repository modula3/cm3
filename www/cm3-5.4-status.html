CM3 5.4 Release Status

This page summarises the current status of the cm3 5.4 release. See the release plan for details.

Getting the source code for this release

You can check out the source code for this release from the release branch via anonymous CVS. 

    cvs -d :pserver:anonymous@birch.elegosoft.com:/usr/cvs login
    [The password is empty, just hit return.]
    cvs -z3 -d :pserver:anonymous@birch.elegosoft.com:/usr/cvs checkout -r release_branch_cm3_5_4 -P cm3

You will need this source tree if you want to install a full cm3 distribution from test installation archives linked below. See the (somewhat outdated) installation notes for details.

There will be no tarball download of the source code for this release until the release is made.
