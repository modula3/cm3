GENERIC INTERFACE CITextTable(Tbl);

 Where Tbl equals Table(Text, Value) 

TYPE
  T = Tbl.T;
  Iterator = Tbl.Iterator;
  Default <: Tbl.Default;

 A CITextTable(Tbl) is a Table(Text.T, Value).T, with the
keyEqual and keyHash methods overriden with case-insensitive
procedures. 


END CITextTable.
