This tool provides the user control over which target machine (C compiler) is used to parameterise M3CBackEnd_C. 

PROCEDURE ToolInit() RAISES {};
Register all the targets as options to the tool. This must be explicitly initialised because the argumeent registration must occur after the targets have registered themselves. This procedure must be called before handling help information. 

PROCEDURE Init(): INTEGER RAISES {};
Initialise the C back-end to the one selected by the user, or the default if unspecified. Returns < 0 if error in target choice. 

CONST  (* M3Args *)
  CCTarget_Arg = "CCTarget";  (* /l *)

END M3CBE_C_Tool.
